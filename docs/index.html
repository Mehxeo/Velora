After installing, open Velora and add your API keys in Settings (OpenAI, Anthropic, Gemini, or DeepSeek). Optional Supabase sign-in enables cloud sync — see the README.

macOS: If you see “Apple could not verify…” / malware wording, that is Gatekeeper for unnotarized builds — Control-click Velora → Open → Open, or open once then use System Settings → Privacy & Security → Open Anyway. For “damaged”, run xattr -dr com.apple.quarantine /path/to/Velora.app or use the .zip, unzip, then Open. A future release can be fully trusted after the project enables Developer ID + notarization in CI.
Windows: If SmartScreen warns, use “More info” → “Run anyway.”
